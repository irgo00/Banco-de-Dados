# Anotações SQL

Este repositório contém anotações e scripts SQL baseados nos principais comandos de bancos de dados abordados em um curso avançado de MySQL na Udemy.

## Sobre
Os scripts aqui armazenados servem como material de consulta e estudo para reforçar conceitos importantes de SQL, abrangendo desde operações básicas até comandos mais avançados.

## Estrutura do Repositório
- `sakila_tiagopassos` - Banco de dados Sakila utilizado no curso.
- `comandos_sql.sql` - Contém todos os principais comandos SQL abordados no curso.
- `tiposDeDados.sql` - Lista e explica os principais tipos de dados utilizados em bancos de dados.
- `atividades.sql` - Atividades propostas durante o curso.
- `Aluguel_anual.xml`, `Aluguel_mensal.xml`, `excel_alugueis_totais.xml` - Arquivos XML utilizados para dashboards.

## Tecnologias Utilizadas
- Banco de Dados: MySQL
- Ferramentas: MySQL Workbench, Visual Studio Code

## Referências
Este material foi baseado no curso avançado de MySQL realizado na [Udemy](https://www.udemy.com/).
